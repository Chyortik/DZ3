##Основные команды Git##
1. git init -  инициализация локального репозитория
2. git status - текущее состояние
3. git add .\<имя файла> - добавление файла или файлов к следующему коммиту
4. git commit - m "message" - создание коммита
5. git log - вывод истории коммитов с их хэш-кодами
6. git clone <url-address repo> - клонирование внешнего репозитория на локальный ПК
7. git pull - направляем локальныую версию репозитория на внешний
8. git push - получение изменений из внешнего репозитория и слияние с локальной версией
9. git branch - проверка места нахождения (в какой ветке мы находимся)
10. git checkout branch_name - переход к ветке
11. git merge - слияние веток с главной или с той, в которой находимся
